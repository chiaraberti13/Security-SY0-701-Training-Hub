## 🇬🇧 English

### Overview
**Security+ SY0-701 Training Hub** is a bilingual study web app for CompTIA Security+ exam preparation. It combines guided theory, objective-based quizzes, and progress tracking in one learning flow.

### Key Features
- **Interactive study manual** organized across the 5 official SY0-701 domains.
- **Objective-based quiz engine** (e.g., 1.1, 2.3) with bilingual content.
- **Detailed explanations** to support understanding, not only memorization.
- **Progress tracking** by objective and completed chapters.

### Quick Start
#### Prerequisites
- Node.js 18+
- npm (included with Node.js)
- Git

#### Installation
```bash
git clone <repository-url>
cd <repository-directory>
npm install
```

#### Run in Development
```bash
npm run dev
```
App URL: `http://localhost:3000`
